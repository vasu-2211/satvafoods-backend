# 🥗 SatvaFoods Backend

SatvaFoods Backend is a Spring Boot application developed for managing an organic food store system. It provides REST APIs for user management, product handling, cart operations, and order processing.

## 🚀 Features

- 👤 User registration and login
- 🛒 Cart management
- 📦 Order processing
- 🛍️ Product management
- 🛠️ Admin functionalities
- 📧 Email service integration

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## 📁 Project Structure

- `controller` → Handles API requests
- `service` → Business logic
- `repository` → Database operations
- `entity` → Database models
